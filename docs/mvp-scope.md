# MVP Scope

## Working Title

Ancient Memphis: A Walk Through the White Walls

## First Slice

A compact, historically scoped district around early Memphis. The exact temple selection is a research task for Step 2, so the first runnable scene uses a provisional temple district layout.

The first slice includes:

- Nile-side arrival point
- mudbrick residential street
- craftsmen area
- temple exterior
- painted interior reconstruction
- ambient NPC routines
- optional narrator mode
- evidence view toggle showing confirmed, inferred, and speculative reconstruction

## Experience Length

The target guided route is 2 to 5 minutes. The initial manifest uses 180 seconds so pacing can be tested early.

## Production Rule

The prototype can be rough geometrically, but the project structure must already support the final promise: cinematic reconstruction with transparent evidence labels and careful provenance.
